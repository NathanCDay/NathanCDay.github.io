Introduction

Below is my plotly plot page submission for the Week 3 project in JHU’s Data Products class. The code below is the Network Graph in R Tutorial from Plotly’s website.

The Data

The data used below is from Wayne W. Zachary’s paper “An Information Flow MOdel for Conflict and Fision in Small Groups”, published in the Journal of Anthropoligical Research. A pdf copy of the original paper can be found here.

The paper looks to explore the mechanisms of fission, by analyzing a university karate club as conflict splits the club into two separate groups over the course of 3 years. The central conflict in this case, is a disagreement between the karate club’s President, John A., and the karate instructure hired by the club, Mr. Hi, over the rising cost of the club’s regular lessons. Club members were divided on the issue of wether to pay Mr. Hi more, which ultimately lead to John A. firing Mr. Hi and a majority fraction of the club resigning and forming a new club under the leadership of Mr. Hi.

The goal of applying network analysis to this social problem, was to be able to accuratly forecast (this paper saw an accuracy rate of 96%) social conflict resolution regardless of culture, by identying sub groups based interaction frequency. The graph below shows the post split interactions among with the John A. and other officers that remained dormant in the lower right quadrant. We can also see that Mr. Hi is not interacting with a lot of remaining club members, instead it looks like Actor 25 (direct connect to Mr.Hi) and Actor 26 (potentially a new leader) act as the major hubs.

The Code

library(plotly)
library(igraph)
library(igraphdata)
library(tidyverse)
library(magrittr)

# get some built in network data
data(karate, package="igraphdata")
G <- upgrade_graph(karate)
L <- layout.circle(G) %>% as.data.frame()
# create verticies and edges
vs <- V(G)
V(G)$label <- names(vs)
edg <- as.data.frame(get.edgelist(G))
num_vs <- length(vs)
# network nodes from L
network <- plot_ly(data = L, x = ~V1, y = ~V2, mode = "markers", colors = "Dark2", text = vs$label, hoverinfo = "text")
# create edge shapes
edg_shapes <- list()
for(i in 1:nrow(edg)) {
    v0 <- edg[i,]$V1
    v1 <- edg[i,]$V2
    edge_shape <- list(
        type = "line",
        line = list(color = "#030303", width = .5),
        x0 = L$V1[v0],
        y0 = L$V2[v0],
        x1 = L$V1[v1],
        y1 = L$V2[v1]
    )
    edg_shapes[[i]] <- edge_shape
}

The Plot

# create network
axis <- list(title = "My First Plotly", showgrid = F, showticklabels = F, zeroline = F)
layout(network,
            title = "Zachary's Karate Data",
            shapes = edg_shapes,
            xaxis = axis,
            yaxis = axis)
