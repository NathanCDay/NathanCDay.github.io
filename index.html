Enable https on GitHub Pages via CloudFlare

Augsut 5, 2017

HTTPS has become the standard for web site for many good security reasons. Unfortunately right now GitHub Pages does not support using a custom domain AND https (the standard domain, username.github.io is automatically served via HTTPS. There is way forward via CloudFlare, an internet service company that is on a mission to make web traffic safer and more efficient.

There a couple of tutorials out there for this, but I used the write up by CloudFlare’s own Juande Ali and highly recommend it. I’m working on a write up for it that will start a new section on here for write ups focused more on web tech and issues outside of data analysis.

Sport Sentiment Analysis

June 1, 2017

I built a scraper for ASAPSports that collects and cleans interview transcripts from a media press event. I then used several different text analysis packages library(sentimentr), library(readability) and library(tidytext) to quantify readability and sentiment of the athletes’ responses. Ultimatley, I discovered that hockey players are slighly nicer than other professional athletes, maybe it’s a Canadian thing eh?
