DC Crime Report

2017/01/29

Introduction

Hi, below is mildly morbid vizualization, I created to map homicides in Washington DC, using color to visualize murder weapon. There is a lot more analysis that I have yet the time to port to Rmd so for now a little plot.

The Data

This data is was sourced from the DC Metropolitan Police Departmentâ€™s ASAP Crime database. The data set I am using here contains only the record of homicides in the District of Columbia for the year 2013, with information about the type of weapon used, time of day, city block.

I have used the ggmap::geocode() function to preprocess the data set and translate the datasetâ€™s original city block information into lattitude and longitude coordinates for this visualization.

sub_crime <- readRDS("~/future/Capital_Crime/data/homicides.RDS")

The Map

offense_pal <- colorFactor(palette = c("#1B9E77", "#D95F02", "#7570B3"),
                           domain = NULL)
leaflet(sub_crime, options = leafletOptions(minZoom = 12, maxZoom = 14)) %>%
    addProviderTiles("CartoDB.Positron") %>% 
    addCircleMarkers(color = ~offense_pal(METHOD), weight = 2, radius = 15,
                     opacity = .5, fillOpacity = .25) %>%
    addLegend("topright", pal = offense_pal, values = ~METHOD,
              title = "Murder Weapon")
